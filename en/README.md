# English Mirror (`en/`)

This directory hosts English translations with path-level mapping to Chinese source files.

## Mapping Rule
- Source: `<module>/<file>.md`
- Target: `en/<module>/<file>.md`

Example:
- `Core/SRT_Core_00_Intro.md` -> `en/Core/SRT_Core_00_Intro.md`
